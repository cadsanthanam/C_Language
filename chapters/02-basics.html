Chapter 2: Basics & Variables

What are Variables?

A variable is a named storage location in memory that can hold data. Think of it like a labeled box where you can store different types of information.

üéØ Key Concept

Variables have three important properties: Name (identifier), Type (what kind of data), and Value (the actual data stored).

Variable Declaration Syntax

data_type variable_name;

Example:

int age;        // Declaration
age = 20;       // Assignment
int height = 175;  // Declaration + Initialization

Data Types in C

C provides several built-in data types to store different kinds of information:

Data Type Size Range Example
int 4 bytes -2,147,483,648 to 2,147,483,647 42, -100, 0
float 4 bytes ¬±3.4E¬±38 (6 decimal places) 3.14, -2.5, 0.001
double 8 bytes ¬±1.7E¬±308 (15 decimal places) 3.14159265359
char 1 byte -128 to 127 (or 0 to 255) 'A', '5', '\n'

Choosing the Right Data Type

  • int: For whole numbers (age, count, scores)
  • float: For decimal numbers with moderate precision
  • double: For decimal numbers requiring high precision
  • char: For single characters or small integers

Variable Naming Rules

Rules (Must Follow)

  1. Must start with a letter (a-z, A-Z) or underscore (_)
  2. Can contain letters, digits (0-9), and underscores
  3. Cannot contain spaces or special characters
  4. Cannot be a C keyword (reserved words)
  5. Case-sensitive (age and Age are different)

Conventions (Best Practices)

  • Use meaningful names: studentAge instead of sa
  • Use camelCase or snake_case consistently
  • Constants in UPPERCASE: MAX_SIZE
  • Avoid single-letter names except for loop counters

Input and Output

Output with printf()

The printf() function displays output to the screen:

#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    printf("The answer is %d\n", 42);
    return 0;
}

Input with scanf()

The scanf() function reads input from the keyboard:

#include <stdio.h>

int main() {
    int age;
    printf("Enter your age: ");
    scanf("%d", &age);
    printf("You are %d years old.\n", age);
    return 0;
}
‚ö†Ô∏è The & Symbol in scanf

Notice the & before the variable name in scanf. This gives scanf the address of the variable so it can store the input there. Without &, scanf doesn't know where to put the data!

Format Specifiers

Format specifiers tell printf and scanf what type of data to expect:

Specifier Data Type Purpose Example
%d int Decimal integer printf("%d", 42);
%f float/double Floating point printf("%f", 3.14);
%c char Single character printf("%c", 'A');
%s string String of characters printf("%s", "Hello");

Formatting Options

printf("%.2f", 3.14159);     // Output: 3.14 (2 decimal places)
printf("%5d", 42);            // Output:    42 (5 characters wide)
printf("%-5d", 42);           // Output: 42    (left-aligned)
printf("%05d", 42);           // Output: 00042 (zero-padded)

Complete Example Program

Let's create a program that demonstrates all the concepts:

#include <stdio.h>

int main() {
    // Variable declarations
    int age;
    float height;
    char grade;
    
    // Get user input
    printf("=== Student Information System ===\n");
    printf("Enter your age: ");
    scanf("%d", &age);
    
    printf("Enter your height (in meters): ");
    scanf("%f", &height);
    
    printf("Enter your grade: ");
    scanf(" %c", &grade);  // Note the space before %c
    
    // Display the information
    printf("\n=== Your Information ===\n");
    printf("Age: %d years old\n", age);
    printf("Height: %.2f meters\n", height);
    printf("Grade: %c\n", grade);
    
    // Some calculations
    int ageInMonths = age * 12;
    printf("You are approximately %d months old.\n", ageInMonths);
    
    return 0;
}

Constants

Constants are values that don't change during program execution:

Using #define

#include <stdio.h>
#define PI 3.14159
#define MAX_STUDENTS 100

int main() {
    printf("Value of PI: %f\n", PI);
    printf("Maximum students: %d\n", MAX_STUDENTS);
    return 0;
}

Using const keyword

const int MAX_SIZE = 50;
const float TAX_RATE = 0.08;

Common Errors and Solutions

‚ùå Forgetting the & in scanf

Error: scanf("%d", age);
Fix: scanf("%d", &age);
Why: scanf needs the memory address to store the input.

‚ùå Wrong format specifier

Error: printf("%d", 3.14); (using %d for float)
Fix: printf("%f", 3.14);
Why: Each data type needs its matching format specifier.

‚ùå Uninitialized variables

Error: Using a variable without assigning a value first
Fix: Always initialize variables: int count = 0;
Why: Uninitialized variables contain garbage values.

‚ùå Character input after numeric input

Problem: scanf("%c", &ch); after scanf("%d", &num);
Fix: scanf(" %c", &ch); (note the space)
Why: The space consumes any leftover whitespace characters.
